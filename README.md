# Alapaka Venkata Ramesh — Portfolio Website

Personal portfolio website built with **HTML, CSS & JavaScript**.

## 🌐 Live Site
[View Portfolio](https://alapakaramesh.github.io/portfolio)

## ✨ Features
- Animated particle canvas hero background
- Typewriter role effect
- Interactive ID badge with 3D tilt
- Scroll-triggered animations (AOS)
- Animated skill proficiency bars
- IBM internship experience timeline
- Foody project showcase
- Education & certifications section
- Contact form + social links

## 🛠 Tech Stack
`HTML5` `CSS3` `JavaScript` `Canvas API` `Intersection Observer API`

## 📁 Structure
```
portfolio/
├── index.html      ← Main HTML
├── style.css       ← All styles
├── script.js       ← Animations & logic
├── profile.jpg     ← Profile photo
└── .gitignore
```
